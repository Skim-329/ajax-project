    About

    Hello! My name is Simon and I am creating an AJAX project to demonstrate my knowledge of HTML, CSS and JavaScript! In addition to these languages, I am requesting information from an API to use for this application. "Wine&Dine" is a wine pairing app which allows you to input a type of food and set a maximum price to discover which wine pairs best with those inputs! Enjoy!
